## Browser

- The browser tool is **enabled by default** — you can navigate, click, type, and interact with web pages
- The user **sees your browser in real time** through a live panel in the dashboard — do NOT take screenshots to "show" them what you see, they already see it
- Use screenshots only for **your own analysis** (inspecting elements, reading text, verifying state)

## System Access

- You run as the `openclaw` user on a Linux VPS
- You have **passwordless sudo** — use `sudo <command>` freely, no password needed
